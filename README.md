# SynthPlus 🎹

**SynthPlus** — это веб-приложение для детализированного хранения, каталогизации и описания ваших synthwave-треков. Приложение предлагает динамический подход к описанию музыки: в зависимости от выбранного поджанра и настроения, вам предлагаются уникальные вопросы, помогающие зафиксировать структуру, атмосферу и особенности звучания конкретного трека.

## Ключевые возможности 🚀

- **Динамический опросник:** Уникальные вопросы для каждого из 17+ поджанров synthwave (Cyberpunk, Darksynth, Retrowave и др.) и 18+ настроений.
- **AI-Промпты в один клик:** Умная генерация англоязычных запросов (prompts) из сохранённых метаданных трека — идеально для использования в Suno, Udio и других нейросетях.
- **Экспорт карточек:** Возможность сохранить красивую "визитную карточку" (SynthPassport) вашего трека в формате PNG или PDF.
- **Надежное хранение:** Легковесная и быстрая база данных SQLite.
- **Поиск и фильтрация:** Удобный архив для навигации по вашей музыкальной библиотеке.

## Технологический стек 🛠

- **Frontend:** React, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express
- **База данных:** SQLite + Drizzle ORM
- **Экспорт:** jsPDF, html2canvas (динамически подгружаемые)

## Локальный запуск (Разработка) 💻

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/liquidsyntes/SynthPlus.git
   cd SynthPlus
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Подготовьте базу данных:**
   Создайте необходимые таблицы в вашем локальном файле SQLite:
   ```bash
   npm run db:push
   ```

4. **Запустите сервер разработки:**
   ```bash
   npm run dev
   ```
   Приложение будет доступно по адресу `http://localhost:5000`.

## Сборка и запуск в Production 📦

Если вы хотите запустить финальную (оптимизированную) версию локально:
```bash
npm run build
npm run start
```

## Информация по Деплою (Хостинг) 🌐

Поскольку приложение использует **SQLite**, база данных физически хранится в файле `sqlite.db` внутри вашего проекта. 
⚠️ **Внимание:** Не используйте ephemeral/serverless платформы (вроде Vercel или бесплатного Heroku) без подключения постоянного "Volume", иначе вы Рискуете потерять свои данные при перезагрузке серверов хостинга.

Лучшие платформы для деплоя этого стека:
1. **Render.com** (с использованием Disk)
2. **Railway.app** (с использованием Volume)
3. Обычный **VPS** (виртуальный выделенный сервер) + PM2
